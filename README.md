# 🚗 Car Affordability Calculator

An Oracle APEX application that calculates the **true total cost of vehicle ownership** going beyond the lease payment to include fuel, servicing, tyres, battery, insurance, depreciation, and more. Built for the Sri Lankan market (LKR).

## Features

- Full monthly cost breakdown across all ownership categories
- Net cost of ownership after estimated 5-year resale value
- Required monthly income calculator (expenditure and equity basis)
- Salary allocation check against total monthly expenses
- **Full Sinhala language mode**  every label and result rendered in සිංහල

## Stack

Oracle APEX + PL/SQL. All calculation logic runs server-side via APEX computation processes. No external dependencies.

## Installation

Import the included `.sql` export file into your APEX workspace. No additional schema objects required.
